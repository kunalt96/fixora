# Fixora SDK

## Installation

npm install fixora-sdk

## Usage

```js
import { init } from "fixora";

init({
  apiUrl: "http://localhost:4000/api", //REPLACE IT WITH BACKEND URL
  apiKey: "your-api-key"
});
```

## Features
- Capture frontend errors
- AI-powered fixes
- Real-time error tracking
- Stack trace analysis
- Confidence scoring
- URL tracking
- Error grouping
- Error count tracking
- Last seen timestamp
